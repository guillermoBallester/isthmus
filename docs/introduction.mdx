## What is Isthmus?

Isthmus is an open-source [Model Context Protocol](https://modelcontextprotocol.io) (MCP) server that gives AI models direct access to your PostgreSQL database. It runs locally as a single binary, connects over stdio, and requires no cloud services.

Point your AI client at Isthmus, and it can discover your schema, understand table relationships, profile data distributions, and execute read-only SQL queries — all with built-in safety guardrails.

## Supported clients

Isthmus works with any MCP client that supports the stdio transport:

- [Claude Desktop](/clients/claude-desktop)
- [Cursor](/clients/cursor)
- [Windsurf](/clients/windsurf)
- [Gemini CLI](/clients/gemini-cli)
- [VS Code (Cline & Continue)](/clients/vscode)

<Info>
  **ChatGPT Desktop** does not yet support MCP's stdio transport. When HTTP transport support lands in Isthmus, ChatGPT Desktop will be supported.
</Info>
